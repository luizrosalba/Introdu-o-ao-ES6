##  Aula 2.2 Functions_Operators 

#### ES 6 Arrow function 
// ES5
var x = function(x, y) {
   return x * y;
}

// ES6
const x = (x, y) => x * y;   /// se colocar mais de uma expressão tem que colocar o return 

const x = (x, y) => { return x * y };  /// boa pratica colocar return sempre
